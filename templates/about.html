About

This web application will provide information on various companies stocks within the last 10 years. This data will be used to help analyze the price of a company's stock over a period of time. Users can visualize this data, predict future trends for a specific company and make comparisons between multiple companies. Our goal was to design make this tool interactive and accessible.

Some of the features include:

  • Retrieving stock data for a company
  • Plotting stock data within a specified data range
  • Exporting plots as images
  • Editing plots and extracting raw data
  • Comparing stocks based on specific attributes

We also included a public API that can be used to request the raw data in our database. This api is capable of taking complex parameters and returning the relevant data in JSON format. The requested data can be used freely for all of your application building and analytical needs.
